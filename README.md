# 🥥 Automated Coconut Sorting Mechanism

![Fusion 360](https://img.shields.io/badge/Fusion%20360-CAD%20%26%20Simulation-orange)
![Arduino](https://img.shields.io/badge/Arduino-Tinkercad-teal)
![Status](https://img.shields.io/badge/Status-Complete-brightgreen)

> Automated coconut sorting mechanism designed to replace manual grading in agricultural packaging lines. Uses size-based mechanical sorting across three grades, validated through CAD modelling, stress analysis, motion simulation, and a working prototype.

---

## 📐 Sorting Grades

| Grade | Hole Size | Description |
|-------|-----------|-------------|
| Grade A | 80 mm | Largest coconuts |
| Grade B | 70 mm | Medium coconuts |
| Grade C | 60 mm | Smallest coconuts |

---

## 🛠️ Tools & Methods

**Fusion 360** — 3D part design, full assembly, stress analysis, and motion simulation.

**Tinkercad** — Circuit design and Arduino programming for automated control logic.

**Prototype** — Physical build to validate mechanical and electronic integration.

## 📊 Key Outcomes

- Mechanical design validated through stress analysis in Fusion 360 — confirmed structural integrity under operational loads
- Three-grade sorting achieved through graduated hole sizing, eliminating manual measurement
- Arduino-controlled actuation automates the sorting sequence end-to-end
- Prototype demonstrated reliable grading performance, confirming design suitability for packaging line deployment
